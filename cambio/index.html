What is Cambio ?

Cambio is a JavaScript library that updates in real-time the UI with the application values.

How to use it ?

Cambio is able to do 4 things: update elements attributes, update text nodes, bind inputs to variables and render templated list items.

Syntax

  • :variable attribute: binds the specified variable in this attribute to the input
  • :items attribute: specifies what list to render
  • {​{ ... }​}: this will be replaced by the evaluation of what is specified inside (variable, short code etc)
    • works in attributes and text nodes

Dynamic variables

You can have dynamic variables, which are constantly evaluated an updated.

$cambio_variable will update variable.

For example, $cambio_applicationData = "JSON.stringify(application)" will update applicationData with this very application object as a JSON string.
